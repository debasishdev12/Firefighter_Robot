On the transmitter side atmega328P arduino bootloader microcontroller is used with 433Mhz rf module.This firefighter
bot has a dedicated remote. On the bot Arduino Mega is used as it provides more pins and the bot has the capability
of doing multiple function.The robot can also be controlled using android apps or by PC using bluettooh technolgy.
A pc side software is developed by C# and is attached to this file.The bot has automatic and manuel mode that
means it can detect fire automatic and throw water on it or by human operator
